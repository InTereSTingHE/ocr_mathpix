# ocr_mathpix
使用mathpix API进行公式/数字/字母识别
<br/>兼顾手写及印刷体
<br/>LATEX, mathml等格式，office软件使用mathml

使用截图工具截图复制图片后运行脚本

MathpixOCR 使用 API keys 连接，[注册获取](https://accounts.mathpix.com/ocr-api.)
<br/>[API doc](https://docs.mathpix.com)
